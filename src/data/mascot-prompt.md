# ExplorePlan Penguin Mascot — Master Character Lock Prompt

Use this prompt at the start of every generation request.

## Character Identity Lock — ExplorePlan Penguin Mascot

A professional SaaS mascot character named "ExplorePlan Penguin". This exact character must remain visually identical in all future generations.

### Core anatomy and proportions (STRICTLY LOCKED)

- Species: cartoon penguin, anthropomorphic
- Head size: large rounded oval, approximately 40% of total body height
- Body shape: smooth rounded torso, slightly wider at bottom
- Arms: short, thick rounded flippers, positioned crossed over chest
- Feet: flat, bright orange, triangular webbed feet, symmetric stance
- Beak: small, rounded triangle shape, bright orange, centered
- Eyes: large oval eyes, black pupils, white sclera, friendly confident expression
- Facial expression: slight confident smile, calm, intelligent, trustworthy
- Pose: upright standing, facing directly forward, symmetrical balance

### Clothing (STRICTLY LOCKED)

- Light-blue modern travel vest
- Vest color: soft sky blue (#7EC3E6 range)
- Vest structure: sleeveless, clean seams, modern minimal design
- Compass pin on chest:
  - Circular shape
  - White face
  - Small red arrow
  - Positioned on upper right chest (viewer left)

### Color palette (DO NOT CHANGE)

- Body black: deep charcoal black (#1E2430 range)
- Belly white: soft neutral white (#F4F6F8 range)
- Beak and feet: warm orange (#F59A23 range)
- Vest: soft light blue (#7EC3E6 range)
- No alternate color schemes allowed

### Art style lock (CRITICAL)

- Clean modern SaaS vector mascot style
- Premium flat vector illustration
- Soft gradient shading (minimal, subtle)
- Smooth rounded outlines
- No sketch lines
- No painterly texture
- No realism
- No anime style
- No exaggerated cartoon proportions
- Professional startup mascot aesthetic

### Rendering rules (STRICTLY FOLLOW)

- Full body always visible
- Perfect centered framing
- Symmetrical balance
- Facing forward toward camera
- White background or transparent background only
- No scene elements unless explicitly requested
- No pose changes unless explicitly requested
- No anatomy changes
- No clothing changes

### Personality conveyed visually

- Intelligent
- Calm
- Trustworthy
- Confident
- Modern SaaS brand identity

### Output format

- 1024x1024 or higher resolution
- PNG
- Transparent background preferred

Always include this:

> no redesign, no different penguin, no different vest, no color changes, no style changes, no anime style, no realism, no sketch style, no exaggerated proportions, no different face, no different eyes, no different beak, no different mascot, no different character
